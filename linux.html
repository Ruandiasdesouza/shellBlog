Linux Kernel

O site Oficial do Linux Kernel é URL: https://www.kernel.org, onde você encontra versões estáveis do kernel Linux, documentação técnica, informações sobre como contribuir para o desenvolvimento do kernel e links para repositórios Git.

Distribuições Linux

Como o Linux é um kernel, ele é usado por diversas distribuições Linux, que são sistemas operacionais completos baseados no kernel Linux. Cada distribuição tem seu próprio site oficial, sendo os mais conhecidos pela comunidade:
Ubuntu: Site: https://ubuntu.com Uma das distribuições mais populares, focada em facilidade de uso.
Debian: Site: https://www.debian.org Conhecida por sua estabilidade e base para muitas outras distribuições, incluindo o Ubuntu.
Fedora: Site: https://getfedora.org Distribuição patrocinada pela Red Hat, com foco em inovação.
Arch Linux: Site: https://www.archlinux.org Distribuição minimalista e altamente personalizável.
Linux Mint: Site: https://linuxmint.com Baseado no Ubuntu, é conhecido por ser amigável para iniciantes.
CentOS: Site: https://www.centos.org Focado em servidores e compatível com o Red Hat Enterprise Linux (RHEL) porém, o CentOS Linux foi descontinuado em 30 de junho de 2024 com o fim da vida útil da versão 7.
openSUSE: Site: https://www.opensuse.org Distribuição poderosa e versátil, com foco em estabilidade e facilidade de uso. Uma recomendação de leitura que dou além da documentação oficial do sistema operacional de sua escolha é o livro, Linux Bible

Terminal, comandos e afins

Neste momento, assumo que você já seja um usuário Linux e queira ser menos dependente da interface gráfica. Por isso, agora vamos ter uma conversa sobre os únicos comandos que você deve se preocupar em aprender a usar neste momento inicial:
man: O comando serve para fornecer informações sobre comandos. Exemplo: man man.
help: O comando ajuda com comandos e opções disponíveis de algum aplicativo, semelhante ao comando man, porém decidi citá-lo separadamente. Exemplo: ls --help.
cd: O comando é usado para navegar pelos diretórios do sistema. Exemplo: cd ./ para voltar à raiz do diretório.
ls: O comando lista os arquivos no diretório atual em que você se encontra. Com esses comandos simples você poderá ter uma autonomia maior, permitindo remover um pouco o medo/dificuldade das linhas de comando que alguns usuários têm. Veja que aqui já dou a chance de você usar o 'pouco' que dei para ler e fazer sozinho. Não dependa 100% do que deixo ou não de escrever. Além do mais por ser uma lista vasta de comandos, não vamos perder tempo escrevendo o básico que se encontra dentro do seu sitema.
